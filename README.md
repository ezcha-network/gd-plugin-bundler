# GD Plugin Bundler

[![Get this plugin on the Godot Asset Store](https://cdn.ezcha.net/static/git-badge/godot-asset-store.svg)](https://store.godotengine.org/asset/ezcha/plugin-bundler/)

This is a simple plugin for Godot that bundles other plugins for distribution on the
asset store.

When enabled the plugin adds a "Plugin Bundler" section in the project settings. Here you
can select the plugin path, a license file to include, and other various options.

To bundle the plugin go to the toolbar and select "Project > Tools > Plugin Bundler..."
